# Установка статус-страницы

Скрипт для установки статус-страницы на сервере.

## Использование

```bash
curl -fsSL -H 'Cache-Control: no-cache' -H 'Pragma: no-cache' "https://raw.githubusercontent.com/Beniamiiin/vpnnode/refs/heads/master/setup-status-page/run.sh?nocache=$(uuidgen)" | sudo bash
```
